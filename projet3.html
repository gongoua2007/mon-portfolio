Projet 3 : Automatisation de tâches avec Python

Présentation du projet

Dans le cadre de ma Licence 1 Informatique, j’ai réalisé un projet d’automatisation consistant à créer un script Python capable de trier automatiquement des fichiers dans différents dossiers selon leur type (images, documents, vidéos, etc.).

Objectifs du projet

  • Découvrir les scripts d’automatisation en Python
  • Utiliser les modules os et shutil pour manipuler le système de fichiers
  • Apprendre à gérer les erreurs et les chemins relatifs
  • Créer une interface simple en ligne de commande

Exemples et captures

Technologies et langages utilisés

Langage : Python

Bibliothèques : os, shutil, time

Outil : Visual Studio Code

Exemple de code Python


import os, shutil

source = "C:/Users/Etudiant/Downloads"
destination = "C:/Users/Etudiant/Organised"

for file in os.listdir(source):
    name, ext = os.path.splitext(file)
    ext = ext[1:]  # retirer le point
    folder = os.path.join(destination, ext)
    os.makedirs(folder, exist_ok=True)
    shutil.move(os.path.join(source, file), os.path.join(folder, file))
print("Tri automatique terminé !")
