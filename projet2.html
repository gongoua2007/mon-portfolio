Projet 2 : Gestion d'une base de données

Présentation du projet

Ce projet a été réalisé dans le cadre de ma formation en Licence 1 Informatique. L’objectif était de concevoir une base de données pour gérer les informations d’une bibliothèque universitaire (livres, auteurs, emprunts, étudiants).

Objectifs du projet

  • Apprendre la modélisation des données (MCD, MLD, MPD)
  • Créer une base de données relationnelle avec MySQL
  • Écrire des requêtes SQL pour insérer, modifier et extraire les données
  • Relier la base à une interface simple en Python / PHP

Illustrations

Technologies utilisées

Langages : SQL, Python, PHP

Outils : MySQL Workbench, VS Code, phpMyAdmin
